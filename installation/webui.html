# Accessing the web interface

Once Komga is started, you can access the web interface at: http://<your-server>:<port>.

Tip

If you are running Komga locally, <your-server> will be localhost.

The default <port> is 8080.

You will be asked for a Login and Password, check the User Accounts section if you haven't done so.
