# Keyboard shortcuts

Key Action
← / ⇞ Previous page (Next page when reading direction is Right to Left)
→ / ⇟ Next page (Previous page when reading direction is Right to Left)
home First page
end Last page
space Sroll down
m Show/hide toolbars
s Show/hide settings dialog
t Show/hide thumbnails view
esc Close book, or close any visible dialog
